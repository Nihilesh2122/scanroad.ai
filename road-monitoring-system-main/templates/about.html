Smart Road Monitoring System

Project Overview

The Smart Road Monitoring System is an AI-powered application designed to automatically detect and classify potholes in road surfaces using computer vision techniques.

This project helps road maintenance teams quickly identify damaged areas, assess their severity, and prioritize repair work efficiently.

Technologies Used

Backend

  • Flask - Web framework for serving the application
  • OpenCV - Computer vision library for image processing
  • SQLite - Lightweight database for storing detections

  • Font Awesome - Icon
  • library

How It Works

Step 1: Input

Upload a road image or use your webcam for real-time detection

Step 2: Detection

OpenCV processes the image using edge detection and contour analysis

Step 3: Results

View detected potholes with severity classification and save to database

Detection Algorithm

The pothole detection algorithm works as follows:

  1. Image Preprocessing: Convert to grayscale and apply Gaussian blur
  2. Edge Detection: Use Canny edge detection to find boundaries
  3. Contour Finding: Identify closed shapes in the edge image
  4. Filtering: Remove noise by filtering based on area and circularity
  5. Classification: Classify severity based on pothole size
  6. Visualization: Draw bounding boxes with severity labels

Future Improvements

  • Deep Learning: Use CNN-based models (YOLO, Faster R-CNN) for more accurate detection
  • Mobile App: Develop a mobile application for field workers
  • GPS Integration: Automatically tag detections with GPS coordinates
  • Cloud Storage: Store images and data in cloud for large-scale deployment
  • API Development: Create RESTful API for integration with other systems
  • Alert System: Email/SMS notifications for high-severity potholes
  • Heat Maps: Visualize pothole hotspots on a map
